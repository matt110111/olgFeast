## 🛠️ **How to Run**

### Super Simple (Recommended)
```bash
# One command to rule them all
./run.sh

# Stop when done
./stop.sh
```

### Manual Setup
```bash
# Backend
cd fastapi_app
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python start_dev.py

# Frontend (new terminal)
cd frontend
npm install
npm start
```
